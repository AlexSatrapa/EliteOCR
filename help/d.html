Trade Dangerous Export
By clicking "Trade Dangerous Export" you will create a "import.prices" file with the result data in your standard export location. You can find this location in Settings. Trade Dangerous Export is a plugin written by gazelle(on frontier dev forum). You can remove this button by deleting TD_Export directory in the plugins directory.

-Back- ======= There are no limitations for export of the data for own usage. However export to BPC and EDDN is limited to following conditions:
- All results must contain system name
- System and station names cannot be identical
- Results cannot be older than 2 hours
- Results must be obtained from screenshots with height bigger than 1080 pixel
